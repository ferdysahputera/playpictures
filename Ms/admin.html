Koneksi Google Spreadsheet

Masukkan Spreadsheet ID dari URL Google Sheets.

Panduan Apps Script
1. Buka Google Spreadsheet
2. Klik Extensions > Apps Script
3. Hapus kode default, paste kode ini:

function doGet(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheetName = e.parameter.sheet || 'Config';
  var sheet = ss.getSheetByName(sheetName);
  if (!sheet) return ContentService.createTextOutput(
    JSON.stringify({error:'Sheet not found'})
  ).setMimeType(ContentService.MimeType.JSON);
  var data = sheet.getDataRange().getValues();
  var result = [];
  for (var i = 1; i < data.length; i++) {
    if (data[i][0] !== undefined && data[i][0] !== '')
      result.push({
        key: String(data[i][0]).trim(), 
        value: data[i][1] !== undefined ? data[i][1] : ''
      });
  }
  return ContentService.createTextOutput(JSON.stringify(result))
    .setMimeType(ContentService.MimeType.JSON);
}

4. Klik Deploy > New Deployment
5. Type: Web app | Execute as: Me | Access: Anyone
6. Klik Deploy, izinkan akses
